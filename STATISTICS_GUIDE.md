# WSwriter 访问统计与访客IP监控指南

## 📊 功能概述

WSwriter 现在集成了完整的访问统计系统，包括：
- 📈 总访问量和浏览量统计
- 🏆 最受欢迎文章排行
- 👥 访客IP统计和分析
- 💬 评论数据统计

## 🔐 访问权限

所有统计数据都受到严格的认证保护：

- ✅ **仅本地访问**：在同一台机器上可以直接查看（127.0.0.1）
- ✅ **已登录用户**：通过 JWT 令牌认证的管理员可以查看
- ❌ **公开禁止**：非管理员或远程访问者无法获取统计数据

## 📱 使用步骤

### 1. 启动 WSwriter

```bash
go run WSwriter.go
```

### 2. 打开统计面板

- 页面顶部导航栏中点击 **"📊 访问统计"** 按钮
- 或者本地访问时会自动展示

### 3. 查看统计数据

#### 📋 概览数据
- **总页面数**：发布的文章总数
- **总浏览量**：所有文章的浏览次数之和
- **评论总数**：全站评论总数
- **独立IP数**：不同访客的IP地址数

#### 🏆 最受欢迎的文章
显示前10个最常被访问的文章：
- 按浏览量排序（降序）
- 显示文章标题和浏览次数
- 点击可快速查看

#### 👥 访客IP统计
显示前50个最活跃的访客IP：

| 列名 | 说明 |
|------|------|
| 🌐 IP地址 | 访客的IP地址（十进制格式） |
| 💬 评论数 | 该IP的评论总数 |
| ⏱️ 首次出现 | 该IP首次评论的时间 |
| ⏱️ 最后更新 | 该IP最后一条评论的时间 |

## 🔍 IP 地址解读

### 本地IP示例
```
127.0.0.1      - 本机回环地址
192.168.1.100  - 局域网内部
```

### 公网IP示例
```
8.141.x.x      - 阿里云用户
203.0.113.x    - 各种ISP
```

## 🛡️ 数据安全

### 隐私保护措施
- ✅ IP 地址**只有认证用户**才能看到
- ✅ 本地环回地址（127.0.0.1）也被保护
- ✅ 数据存储在**本地文件系统**，不上云
- ✅ 所有 API 请求需要 **JWT 认证**

### 防止数据泄露
| 操作 | 权限检查 |
|------|----------|
| 查看统计 | ✅ 本地或已认证 |
| 查看IP | ✅ 本地或已认证 |
| 导出数据 | ✅ 本地或已认证 |
| 删除数据 | ✅ 仅本地 |

## 📊 API 端点

### 获取整站统计
```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  http://localhost:8080/api/statistics
```

**响应格式**：
```json
{
  "success": true,
  "data": {
    "total_pages": 12,
    "total_views": 456,
    "total_comments": 23,
    "pending_comments": 2,
    "unique_ips": 15,
    "pages": [
      {
        "path": "content/zh-cn/post/example/index.md",
        "title": "示例文章",
        "views": 45
      }
    ],
    "visitors": [
      {
        "ip": "192.168.1.100",
        "comment_count": 5,
        "first_seen": "2026-03-25 10:30:00",
        "last_seen": "2026-03-30 15:45:00"
      }
    ]
  }
}
```

### 获取单页面统计
```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "http://localhost:8080/api/page_statistics?path=content/zh-cn/post/example/index.md"
```

## 💡 实用建议

### 识别活跃读者
- 查看评论数最多的IP地址
- 这些通常是你的忠实读者或评论者

### 了解受众分布
- 观察独立IP数的增长趋势
- 判断内容的传播范围和影响力

### 发现热门内容
- 查看最受欢迎的文章列表
- 了解哪些话题最吸引读者

### 垃圾评论检测
- 如果同一IP短时间内发大量评论，可能是垃圾
- 可以将其IP添加到黑名单（在评论设置中）

## 🐛 故障排查

### 问题：显示"无权查看统计数据"
**原因**：未登录或令牌过期
**解决**：点击"登录"按钮，输入管理员凭证

### 问题：IP地址显示为空
**原因**：旧评论没有记录IP
**解决**：只有新提交的评论才会记录IP

### 问题：统计数据不更新
**原因**：缓存或数据未刷新
**解决**：点击"🔄 刷新数据"按钮
