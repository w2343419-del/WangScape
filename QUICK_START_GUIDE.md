# WSwriter 快速使用指南

## 🚀 启动系统

```bash
d:\hugo\WSwriter.exe
```

系统会自动在浏览器中打开管理界面: http://localhost:8080

## 📝 核心功能使用

### 1️⃣ 创建新文章

1. 点击左侧 **"+ 新建文章 (双语同步)"** 按钮
2. 输入中文标题（如：`我的第一篇文章`）
3. 输入分类（如：`生活,随笔`）
4. 点击 **"创建"**

✨ 系统会自动：
- 翻译标题为英文
- 创建中文版本 (zh-cn)
- 创建英文版本 (en)
- 在列表中显示两个版本

### 2️⃣ 编辑文章

**方式1: 直接编辑**
- 在文章列表中点击编辑按钮
- 修改内容
- 按 **Ctrl+S** 或点击 **保存** 按钮

**方式2: 编辑中文后自动翻译**
- 编辑中文版本并保存
- 系统自动生成翻译并更新英文版本
- 看到提示 "✅ 已保存并同步翻译"

**编辑器工具：**
- 🔗 **插入链接**: Ctrl+K
- **B** **加粗**: Ctrl+B  
- *I* **斜体**: Ctrl+I
- 📝 **代码块**: 点击工具栏按钮
- 🖼️ **插入图片**: 点击工具栏按钮
- 📊 **插入表格**: 点击工具栏按钮

### 3️⃣ 删除文章

1. 在文章列表中找到要删除的文章
2. 点击右侧的 **🗑️** (垃圾桶) 按钮
3. 确认删除提示

⚠️ **注意**: 删除操作不可恢复！

### 4️⃣ 查看评论

1. 进入 **📋 待审核评论** 视图
2. 查看所有未审核的评论
3. 看到评论者的 IP 地址和 User-Agent

### 5️⃣ 管理评论

#### 批准评论
- **单个批准**: 点击评论下的 **✅ 批准** 按钮
- **批量批准**:
  1. 点击 **☑️ 全选** 复选框
  2. 点击 **✅ 批量批准**

#### 删除评论
- **单个删除**: 点击评论下的 **🗑️** 按钮
- **批量删除**:
  1. 勾选要删除的评论
  2. 点击 **🗑️ 批量删除**

⚠️ **注意**: 删除评论不可恢复！

#### 导出评论
1. 进入 **📋 待审核评论** 视图
2. 点击 **📥 导出评论 CSV**
3. 文件会下载为 CSV 格式，可用 Excel 打开

### 6️⃣ 配置评论系统

#### 邮件通知设置

1. 进入 **📋 待审核评论** 视图
2. 在右侧面板的 **⚙️ 评论设置** 中：
   - ✅ 勾选 **启用SMTP**
   - 输入 SMTP 服务器地址（如: smtp.gmail.com）
   - 输入 SMTP 端口（通常 587）
   - 输入用户名和密码
   - 输入发件人地址
   - 输入接收通知的邮箱地址

3. 点击 **💾 保存设置**

当有新评论时，系统会自动发送邮件通知！

#### 黑名单设置

**IP黑名单:**
1. 在 **⚙️ 评论设置** 中找到 **IP黑名单**
2. 每行一个 IP 或 IP 前缀（支持通配符）：
   ```
   192.168.1.1
   10.0.*
   127.0.*
   ```
3. 点击 **💾 保存设置**

**关键词黑名单:**
1. 在 **⚙️ 评论设置** 中找到 **关键词黑名单**
2. 每行一个关键词：
   ```
   spam
   advertisement
   casino
   ```
3. 点击 **💾 保存设置**

匹配黑名单的评论会自动被拒绝！

## 🔧 常见问题

### Q: 保存后文章没有更新怎么办？
**A:** 页面会自动刷新。如果没有看到，按 F5 刷新页面。

### Q: 删除了评论想恢复怎么办？
**A:** 遗憾的是，删除的评论无法恢复。操作前请确保无误。

### Q: 邮件为什么收不到？
**A:** 检查以下几点：
1. SMTP 设置是否正确
2. 邮箱账户是否允许第三方应用
3. 防火墙是否阻止了 SMTP 端口

### Q: 翻译为什么不准确？
**A:** 系统使用简单的翻译方法，对于复杂文本可能不够准确。
建议：手动编辑英文版本以获得最佳效果。

### Q: 支持多少个编辑者同时编辑？
**A:** WSwriter 是单人编辑工具，不支持多人同时编辑。
